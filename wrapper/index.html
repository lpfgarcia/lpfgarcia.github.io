Noise filtering using decomposition strategies

Improving the performance of noise filters in multi-class classification problems: an scheme based on One-vs-One decomposition strategy

Authors: Luís P.F. Garcia, José A. Sáez, Julián Luengo, Ana C. Lorena, André C.P.L.F. de Carvalho and Francisco Herrera

Abstract: Noise filters are preprocessing techniques whose goal is to improve data quality in a classification problem by detecting and eliminating examples that contain errors or noise. However, filtering can also remove correct examples and examples containing valuable information that could be useful for the classification task. Therefore, there is usually a margin of improvement on noise detection accuracy for almost any noise filter. In multi-class classification problems, which have more than two classes to be distinguished, binary decomposition strategies have been successfully used to improve the performance of classifiers, by decomposing the problem into several simpler binary subproblems. This paper proposes a noise filtering scheme based on the use of One-vs-One decomposition strategy to improve the noise detection accuracy of noise filtering techniques. Therefore, we adapt the idea on the use of decomposition strategy to noise filtering in multi-class problems in order to make make the noise identification simpler and more accurate. The experimental results show that the One-vs-One decomposition strategy increases the performance of the noise filters studied, detecting more accurately the noisy examples in multi-class classification tasks.
